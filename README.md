# Homebrew Tap — Codihaus

Custom Homebrew tap for [Codihaus](https://github.com/codihaus) projects.

## Install

```bash
brew tap codihaus/tap
```

## Available Casks

| Cask | Description |
|------|-------------|
| `devports` | Tiny macOS menu bar app for monitoring local dev servers and ports |

### DevPorts

```bash
brew install --cask codihaus/tap/devports
```

See [codihaus/devports](https://github.com/codihaus/devports) for details.
